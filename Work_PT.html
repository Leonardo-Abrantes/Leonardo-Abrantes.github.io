Alien Invasion

Versão personalizada do “Alien Invasion”, um jogo muito popular que tem a finalidade de um jogador lutar contra alienígenas que tentam assumir o controlo da Terra.
Esta versão tem algumas semelhanças com a versão original mas com características próprias, mantendo no entanto o objetivo original que é o jogador movimentar uma nave com o intuito de não permitir que os inimigos entrem no planeta Terra e destruam a humanidade.

Run And Eat

Clique duas vezes para ver

O desenvolvimento deste projeto teve como motivação a promoção de um estilo de vida saudável e o impacto que a gamificação pode ter neste objetivo. Desta forma, foi desenvolvido o jogo 2D “Run and Eat”, através da ferramenta Unity para o sistema operativo Windows, com o objetivo de sensibilizar as pessoas para os problemas causados por um estilo de vida sedentário, aplicando os incentivos que a gamificação oferece.

Escape the System:The Missing Parts

Clique duas vezes para ver

Num mundo onde o ensino tradicional se torna cada vez menos efetivo, pretende-se com este projeto apresentar uma abordagem alternativa, que tem em consideração a motivação e o envolvimento dos alunos no ensino. Desta forma, foi desenvolvido o jogo sério 3D chamado “Escape the System: The Missing Parts”, através da ferramenta open-source Unity para o sistema operativo Windows, que tem como objetivo ensinar o funcionamento dos vários componentes que constituem um computador e demonstrar o impacto que a utilização da gamificação e do tipo de jogo escape room podem ter nesse processo.
