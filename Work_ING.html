Mobile Sign Language Recognition System

Previous Next

Since the majority of Portuguese people don't know Portuguese Sign Language (LGP), communication between deaf people and hearing people is still quite complicated, so the existence of a portable system accessible to everyone that recognises LGP gestures and classifies them automatically would greatly facilitate communication between all Portuguese people.

Therefore, the aim of this project was to develop a system based on LGP that would satisfy the points mentioned above. However, due to the lack of image datasets with Portuguese Sign Language gestures and due to the lack of time and resources to create one from scratch, it was necessary to adapt this project and develop a system directed to American Sign Language (ASL).

That said, a model was created and trained using Deep Learning, capable of interpreting an image as a human would, i.e. when faced with an image, it is able to identify which American Sign Language gesture is in question or that the image has no valid gesture.

This model was later incorporated into an Android mobile application that was developed to enable gesture recognition dynamically, i.e. in real time by analysing the frames received by the device's front or rear camera, and gesture recognition statically, i.e. through a photo taken directly from the application or an image in the device's gallery, in a way that is intuitive for users through the graphic interface developed for it.

The developed application can be used completely offline and has as its only requirement that the device use the Android operating system.
